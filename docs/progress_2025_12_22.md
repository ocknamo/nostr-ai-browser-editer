# NABE 開発進捗レポート

## 📅 作業日: 2025/12/22

## 🎯 プロジェクト概要

**NABE (Nostr AI Browser Editor)** は、GitHub上のフロントエンドリポジトリをStackBlitzでプレビューしつつ、Nostr経由でAI指示を送信し、GitHub Actionsで修正を反映する反復開発ワークフローツールです。

---

## ✅ 完了した作業

### 1. ブラウザアプリ基盤構築

#### 技術スタック
- **フレームワーク**: Svelte 5 (Runes API)
- **ビルドツール**: Vite
- **Nostr**: rx-nostr + nostr-tools
- **プレビュー**: StackBlitz SDK

#### 実装したコンポーネント
| コンポーネント | ファイル | 機能 |
|--------------|---------|------|
| App | `src/App.svelte` | メインアプリケーション、状態管理 |
| Header | `src/lib/components/Header.svelte` | ヘッダー、ナビゲーション、メニュー |
| ChatView | `src/lib/components/ChatView.svelte` | AI指示入力、メッセージ履歴 |
| PreviewView | `src/lib/components/PreviewView.svelte` | StackBlitzプレビュー表示 |
| Message | `src/lib/components/Message.svelte` | チャットメッセージ表示 |

#### Nostrクライアント
- `src/lib/nostr-client.ts`: Nostr通信ライブラリ
  - 秘密鍵の自動生成・保存
  - NIP-07対応（ブラウザ拡張）
  - AI指示イベント送信 (kind: 30101)
  - 質問回答イベント送信 (kind: 30103)
  - 購読・イベント受信

---

### 2. UIレイアウト改善

#### ヘッダー最適化
- 高さを **50px** に縮小
- コンパクトなデザインでプレビュー画面の縦スペースを最大化
- リポジトリ情報を `owner/repo:branch` 形式で表示
  - GitHub URLも自動的に `owner/repo` 形式に変換

#### ハンバーガーメニュー
- Repository/Branch設定フォームをメニュー内に移動
- npub表示とコピー機能

#### プレビュー画面
- フォームを削除し、ボタンのみ表示
- Load/Reloadボタンを1つに統合
  - 初期状態: 「▶ Load Preview」
  - ロード中: 「⏳ Loading...」
  - ロード後: 「🔄 Reload Preview」
- ボタンは横幅100%で表示

#### チャット画面
- フォームを削除（メニューに集約）
- シンプルなUI

---

### 3. StackBlitzプレビュー機能

#### 修正した問題
1. **display: none問題**
   - `class:visible` ディレクティブが正しく機能しない問題を修正
   - `style:display` インラインスタイルに変更

2. **iframeの高さ問題**
   - 500px固定から画面いっぱいに表示するように修正
   - Flexboxレイアウトで残りスペースを埋める
   - `:global(iframe)` セレクタでサイズを強制

#### 機能
- GitHub リポジトリの入力（URL形式も対応）
- StackBlitz WebContainersでプレビュー表示
- リロード機能

---

### 4. GitHub Actions テンプレート

#### ファイル構成
```
.github/
├── template/
│   ├── apply-instructions.yml  # AI指示実行ワークフロー
│   └── authorized_npub.example # npub設定例
```

#### ワークフロー機能
- 2分ごとのNostrイベントポーリング
- npub認証による権限チェック
- Aider (AI CLI) による自動修正
- Git操作 (commit, push)
- PR作成/更新
- 完了イベント送信 (kind: 30104)

## 🚀 使い方

### 開発サーバー起動
```bash
cd nostr-ai-browser-editer/browser-app
npm install
npm run dev
```

### ブラウザでアクセス
http://localhost:5173/

### 操作手順
1. ☰ メニューを開いてRepository/Branchを設定
2. **Chat**タブ: AI指示を入力して送信
3. **Preview**タブ: 「Load Preview」でStackBlitzプレビューを表示
